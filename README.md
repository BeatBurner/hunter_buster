## Installation

Install the tool

```cd $HOME
wget https://raw.githubusercontent.com/BeatBurner/hunter_buster/master/install.sh
chmod +x install.sh
bash install.sh
```
 ## After Installation
```
h-buster
```
## How to uninstall
```
pkg uninstall hunterbuster
```
